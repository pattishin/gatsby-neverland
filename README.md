# gatsby-neverland

Based on [Gatsby documentation - Building with Components](https://www.gatsbyjs.org/docs/building-with-components/).

## Install

Make sure that you have the Gatsby CLI program installed:
```sh
npm install --global gatsby-cli
```

Run it by:
```sh
npm install && gatsby develop
```
